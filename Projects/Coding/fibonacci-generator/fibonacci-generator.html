Fibonacci Generator

I made this because I was looking for projects to work on so I can get better at C and saw some youtubers make this so I thought to give it a shot.



Code down below


#include <stdio.h>
#include <unistd.h>
        
int a = 1;
int b = 0;

int main() {
    for (;;) {
        int c = a + b;
        b = a;
        a = c;
        printf("\n%d", c);
        sleep(1);
    }
}
